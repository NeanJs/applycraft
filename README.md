## 🎯 What is ApplyCraft?

ApplyCraft helps job seekers customize resumes for specific job postings using AI.

Paste your resume. Add a job description. Get:

* ATS Match Analysis
* Keyword Suggestions
* Resume Improvements
* Cover Letter Generation
* PDF Export

## 🚀 Getting Started

```bash

cd applycraft

npm install

npm run dev
```

Open:

```text
http://localhost:3000
```
